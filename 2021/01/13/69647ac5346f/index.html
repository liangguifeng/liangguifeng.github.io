2019年工作总结

2019年总结

前言

转眼时光飞逝，毕业都快一年，这一年呢我感觉应该是人生中比较有意义的一段。先从毕业开始聊起吧，我是去年3月底孤身一人来到深圳这个陌生的城市的，当然南方人回到南方也不算陌生，因为天气原因适合自己，所以也倍感亲切，但是回想当初自信满满的从学校出发，到深圳来认知自我来了，哈哈哈哈，还别说，深圳的技术氛围确实很浓厚。

我的工作经历

第一次社招面试的简单经历

当初初来乍到深圳，确实压力很大，首先吧，自己即将步入社会，从学生到一个 PHPer，是满怀期待的，但是事情岂有如此容易，我第一份工作面试了差不多10来家公司，最后收到 offer 的也才三家，原因嘛，很简单，当时基础太差了（其实现在也不咋地，但是有心向学），其他的面试大多数平平淡淡，问些百度都能查到的一些面试题。
比较印象深刻的应该是某某公司吧（太久了，我也记不起来，我只记得是在前海湾那一块的一家公司），面试过程很随和，就是不会紧脏知道吧（要知道我每次面试其实都很紧张0.0，我也不知道我自己在紧张啥），和老板聊了两个多小时吧，其实他们就是想找个人打打下手，写写测试，改改页面的，老板也是明说了，不要求我能力有多好，交际能力好就行，平时跟着其他老员工多学学（其实就是端茶倒水就完事了），就这样，我收到了这家公司的 offer。
然后接下来一天我就去面试了最后一家公司，那就是我最后实习的公司，石方数链了，啊哈，说到石方肯定不能忘的就是财哥了，财哥和林子姐当时面试的我，其实财哥应该是考虑到了我是个应届生，没有问我太过于深层次的东西，我感觉面试就是在聊天一样（虽然我也有很多问题没答上来。。但是我都用了自己的理解去回答了），可能是因为我当时交流还比较通畅，财哥看出了我的爱学，哈哈哈哈哈哈哈哈哈（其实是我自己夸自己，我哪有啥优点~），然后财哥破例收了我。
当时回去晚上我还在想，是去哪，因为我收到了三个offer。

  1. 第一家吧，平淡无奇，老板的面试让我无比紧张，框架用的是ThinkPHP ，办公司地点在深圳北那边（说实话我还真不在乎工作地点远近）
  2. 第二家吧，应该会是特别轻松，因为老板也说了，就是招打杂的，平时写写测试用例，学学代码规范然后看代码就行了。
  3. 第三家吧，因为财哥独特的人格魅力（嘻嘻，马屁一下），外加上公司整体的技术氛围，真的很棒~

我仔细一想，我岂是那种平淡无奇的人，然后就是打杂，想一下，如果我只想打打杂，又何必孤身一人来到深圳呢？我可是有梦想的人，然后果断去了石方

石方数链

去了公司的第一天，糗大嘞，项目跑了一天没跑起来，哈哈哈哈，因为当时我们用的框架是 Laravel ，我在大学自学的时候，也是看的 Laravel 的教程，所以还是了解一点点的，但是由于当时项目包含了前端的一些页面，需要 npm,yarn 等安装才能看到项目的界面，我当时哪懂这些东西，我还傻傻的以为 PHP后端工程师 只需要会 PHP 呢（嗐，其实真正纯做PHP后端的也可以不写前端，但是起码你得懂），后面省略太多过程了，就是一些经常性问一些没有营养的问题的东西，但是还别说，因为性格开朗的原因，比较容易融入团队，很快就和公司其他人沟通起来了，最早带我的人名字叫 小勇 ，小勇是特别有耐心，而且特别好学的一个人，当初我因为技术太差，差点要收拾收拾走人了，是小勇一路带着我，让我逐渐有了起色，然后当时因为基础确实不行，搞不了那个中台项目，后面调到了好赛擂这个项目这边，跟着 十一哥 混饭吃了，十一哥呢为人比较幽默，当时不少没有营养的问题都是问的十一哥，当时在实习的时候，因为 SQL 基础不行，十一哥特地给我找了教程，我恶补了一段时间 SQL，后面因为一个问题，说实话，十一哥可能是对我有点失望了，就是我当时以为 if 语句的大括号是个闭包。。所以我像平时闭包引用外部变量的时候一样使用 use 来引入变量，嗐，因为这一点，我可是有点惨了，下定决心恶补 PHP 基础，就这样，那时候每天早早下班回来，就是为了多看点基础的东西，每天搞到半夜两三点，逐渐有了起色，后面因为想换办公地点了，我们搬到了长虹科技大厦，那一段时光学了不少好东西，比如设计模式呀，代码规范呀，版本控制啊，哎，三言两语道不清一路的过程，就省略了吧，最后离开石方是因为我毕业了，我觉得我也应该出去走走看看别处的世界，然后我离职了

尊一互动

第二家公司是奇学，尊一旗下的子公司，公司规模不算大，但是总公司名号大呀，上市公司，不选了，就这吧~ 到了这，刚开始那肯定少不了的先融入公司，这里也是告诫所有的学弟学妹们要学会融入集体，毕竟工作不是学校，只有融入集体才能开开心心干活，当时公司主要是依赖微信的生态写小程序的，做的是教资方向，当时的后端就是 鑫哥啦，鑫哥的基础我感觉很牢固，不像我松松散散（毕竟我是个弟弟，叫我坤坤，哈哈哈哈哈哈哈哈哈）当时在奇学是我感觉我们技术团队比较好玩的，首先鑫哥和天哥日常撕逼（哈哈哈，天哥是做前端的，也是我学 vue 的导师，当然学 vue 跑不了要学JavaScript了，我借着空余时间，把JavaScript的基础刷了一遍，然后借着空余时间，帮天哥写点小东西，锻炼一下自己，两个经常为了接口的事情撕逼，但是绝对不是真生气，都是闹着玩的），就这样，其乐融融，在奇学偷偷学了一点点前端技术，可惜呀，好景不长，因为我们这个子公司不赚钱，总公司不打算继续把我们这个项目做下去了，然后就解散了（还拿了补偿，算是很良心的公司了），虽然目前和天哥鑫哥不在一家公司了，但是还是会经常聊天，聚在一起喝点小酒，吹吹牛逼~

墨兔科技

墨兔就是我现在所在的公司啦，首先那得是我的老大呀 – 荣哥，荣哥很强，起码够我学的，我有太多的东西要学了，刚进来先是扩展包开发，扩展包开发，说实话，我只看文档里说过，还没有用过，至于扩展包开发的原理，这时候就得去看 composer 的相关东西了，目前我的博客也是扩展包开发，管理后台依旧为了方便使用 laravel-admin，然后就是代码翻阅，很多写法都是我没有想到的，又增进了一步，然后就是 cloud-init 了，一个系统初始化的脚本，能在系统初始化完毕后，自动执行脚本的命令，很是方便，特别是对于公司而言。然后我在周末空余时间，又自学了 docker，docker-compose 等运维相关的东西，其实我对这些东西贼感兴趣，甚至有时候为了研究这些东西熬到大半夜。目前我还在看 Kubernetes 的一些东西，人总是在自己未知的领域特别感兴趣，我也一样

工作总结

我从2019年的一个应届毕业生，到目前的刚入门 PHPer，有很多多的想法和感悟，首先在19年 PHP 的基础算是告一段落了，但是并不代表精通，也就是说我遇到问题我知道该如何去寻找答案了~，而不是问一大堆没有营养的问题，这会使人心生厌烦，然后就是熟悉了 laravel, thinkphp 等框架，再到 JavaScript,vue,bootstrap等前端一些相关的知识，虽然懂的不多，但是起码不是一问三不知了，再然后就是运维方向的了，因为我 Linux 底子不错（因为大学学得好，Linux课堂老师对我特好），所以学 docker,cloud-init,shell 也能够看得懂，虽然只是冰山一角，但是起码有了好的开始，坚持住，我肯定相信我自己没问题的啦（我这么帅气的人会有问题？），接下来在空余时间看看 Kubernetes 的教程，继续探索 PHP 更深层次的东西，人嘛总是要折腾一下子，不然多没劲，技术这东西，总是知道的越多，不懂的也就越多，虽然我知道的不算多，但是门开了，我知道该如何走下去，也告诉学弟学妹们不要担心自己学不会，更不要囫囵吞枣，文档要仔细的看，实在静不下心来就默读，多读几遍，用脑子思考，碰到难题时不要退缩、烦躁，平静下来，梳理思路，一点一点整理，实在不行一段一段打注释，不要因为一些困难就放弃自己前进的路！
